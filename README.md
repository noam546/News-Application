## Getting Started

### Prerequisites

- Java Development Kit (JDK) 11 or higher
- Maven
- Node.js and npm

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/news-application.git
   cd news-application
   npm install
	```
2. Set up the Server:
	```bash
   cd server
   mvn clean install
	```
3. Set up the Client:
	```bash
   cd client
   npm install
	```

### Starting the application 

1. Starting the Serve:
	```bash
   cd server
   mvn spring-boot:run
	```
2. Starting the Client:
	```bash
   cd client
   npm run dev
	```
